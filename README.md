# Pinyin Server

NodeJS Pinyin Server

## DEBUG

```sh
npm i

# socketio
npm start &
node src/client.js

# ws
node src/ws.js &
node src/client-ws.js
```

## Usage

Make sure you have NodeJS 12+ installed.

Open a terminal and type:

```
npm i
npm start
```

## Bundle executable

Open a terminal and type:

```
npm run build
```

Then copy the files in `out/`
